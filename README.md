# テトリス練習ツール

ぷよぷよテトリスのテトリス（ガイドライン／SRS）をベースにした、**練習特化**の単一構成 Web アプリ。
ビルド不要・依存なし。`index.html` をブラウザで開くだけで動きます（`file://` 直開きも可）。
**PWA対応**（オフライン起動・ホーム画面に追加可）。ローカル確認用に `serve.ps1` を同梱。

## 主な機能

- **SRS 準拠の操作（ぷよぷよテトリス2 互換）**：7-bag、ホールド、ゴースト、ソフト/ハードドロップ、
  壁蹴り（I ミノ専用テーブル含む）、**単発の180°回転**。回転形状・壁蹴り・出現位置/向きは
  公式SRSと突き合わせて検証済み（後述）。
- **T-Spin 判定（役の検出）**：ガイドラインの 3-corner ルールで T-Spin / T-Spin Mini を自動判定し、
  消去ライン数と合わせて「T-Spin Double」等を表示・カウント。TSTキック格上げにも対応。
- **練習モード**
  - **フリー**：自由に積んで指慣らし・自由研究。
  - **手順型ドリル**：1手ごとに置き場所を**黄色のヒント**で提示。目標と違えば警告。検証済みの基礎/PCドリル。
  - **完成形テンプレ（field型）**：**うすい色で目標の形を表示**し、その形を組み上げれば成功。
    スピン・ホールド・ソフトドロップOKなので、Tスピン系・各種積みなど**どんなセットアップでも表現可能**。
  - **掘り(Dig)**：お邪魔ブロックを掘って消す中盤・対応力の練習。
  - **自動反復**：完了すると自動で最初からやり直し（設定でON/OFF）。
- **名前付きセットアップ・カタログ**：開幕／中盤・ドネイト／パフェ系の各テンプレ名を分類して一覧化。
  盤面が登録済みの枠は即練習、未設定の枠は「組んで保存」で有効化（後述）。
- **主要セットアップの初期データ同梱**：PCO・開幕TSD(TKI-3)・DT砲・BT砲・ガムシロ積み・はちみつ砲・
  アルバトロス・合掌TSDパフェ・Antifate・QT砲・ダブルダガー・千鳥格子 など計34設定を初期収録。
  - うち**22設定は検証済みの完成形(スケルトン)**を同梱し、選ぶだけで即練習（「✓ 練習可」表示）。
  - 残りは**推奨テト譜コードを同梱**（「推奨テト譜あり」表示）。選ぶと入力欄に自動セットされ、
    「完成形/手順で登録」で取り込めます。
  - 出典: [swng/opener_db](https://github.com/swng/opener_db)（Ivan(28283) Comprehensive Opener Database）。
- **盤面エディタ（保存/ビルダー）**：未設定枠を選ぶ→フリーで形を組む→「現在の盤面を保存」で、
  その正確な形をテンプレとして登録。**自作テンプレ**も新規保存できる。データは `localStorage` に永続化。
- **テト譜(fumen)取込**：`v115@…` / `v110@…` のコードを貼り付けて登録。2通り選べる：
  - **完成形で登録**：最終ページの形を field 型テンプレに（自由に組んで完成形を目指す）。
  - **手順で登録**：複数ページから**1手ずつの手順**を自動抽出し、**各手の置き場所をヒント表示**する
    手順型(fsteps)テンプレに。スピン/tuck を要する設置もテト譜の最終位置に合わせれば正解。
  標準実装（knewjade/tetris-fumen）準拠のデコーダを内蔵し、ランダム3000件で本家と完全一致を確認済み。
- **インポート / エクスポート**：テンプレ（保存盤面）を JSON で書き出し／取り込み。端末間の共有・バックアップに。
- **一手戻す（Undo）**：直前の操作（設置・ホールド）に戻れる。何手でも遡れる。
- **設定**：ゴースト／ヒント／重力（自動落下）／自動反復のON/OFF。
- **手順型ドリルの検証**：盤外・重なり・穴・PC成立を自動チェックして表示。

## 操作

| 操作 | キー | タッチ |
|---|---|---|
| 移動 | ← → | ← → |
| 右回転(CW) | ↑ または X | ⟳ |
| 左回転(CCW) | Z または Ctrl | ⟲ |
| 180°回転 | A | 180° |
| ソフトドロップ | ↓ | ↓ |
| ハードドロップ | Space | ドロップ |

## 回転システムの検証（ぷよぷよテトリス2 互換）

ぷよテトはガイドライン準拠の SRS を採用しているため、本ツールの回転を公式SRSと
独立実装で突き合わせて検証しています。

- **ミノ形状**：7種×4回転=28 形を本家 `tetris-fumen` の `getBlocks` と照合 → 全一致。
- **壁蹴り（CW/CCW）**：SRS 公式オフセット法を独立実装し、ランダム盤面で全ミノ・全遷移・
  左右回転を約 5 万ケース比較 → 全一致（Iミノ専用テーブル含む）。
- **出現位置/向き・7-bag**：ガイドライン（I=3〜6 列, O=4,5 列, JLSTZ=3〜5 列）に一致。
- **O-Spin（O 回転）**：ぷよぷよテトリス2 に合わせ、O も回転で 2×2 が中心まわりに移動し、
  標準SRSの壁蹴りで O-spin が可能（`cambridge-modpack「SRS O-Spin」`方式）。スポーン形は従来どおり。
  ※ PPT2 内部の正確な O オフセット表は公開資料が無いため、標準的な O-Spin 方式を採用。
- **180°回転**：ぷよぷよテトリス2 の単発180°に合わせて実装。PPT2 内部の正確な180°キック
  テーブルは公開資料が無いため、現行ガイドライン系で標準的な180°キックセットを採用
  （`engine.js` の `KICKS_180`。開幕空間ではその場で反転、床/壁際は蹴りで成立）。

## 名前付きセットアップの登録方法（カタログ枠を埋める）

DT砲・各種パフェ積み・ドネイト等の名前付きセットアップは、正確さを担保するため
**実際の形を登録して使う**方式です（不正確な配置を自動生成しません）。手順：

1. 右の一覧から、埋めたいテンプレ枠（「未設定→組んで保存」表示）を選ぶ。
2. 次のいずれかで形を入れる：
   - **テト譜から**：その設定のテト譜コード（`v115@…`）を貼って「テト譜から登録」。最終ページの完成形が入る（最速）。
   - **手で組む**：盤面エディタ（フリー操作）でその形を組み、「現在の盤面を保存」。
3. その枠が「✓ 練習可」になり、完成形ターゲットとして反復練習できる。

- 一度作れば `localStorage` に保存され、次回も使えます。
- **エクスポート**でJSONを書き出し、別端末で**インポート**すれば共有・バックアップできます。
- 全く新しいテンプレは「新規テンプレとして保存」で名前を付けて登録できます。
- ※ 盤面エディタ中はライン消去しないので、揃った行を含む形もそのまま保存できます。

## 手順型ドリルの追加方法（コード）

`js/templates.js` の `list` に `steps` を追加するだけ。各手は
`{ piece, rot, col }` で表し、`col` は「そのミノが実際に占める一番左の列」（0=左壁）。
落下位置（行）はエンジンがその時点の積みから自動計算するため、
人間の操作「回転 → 列へ移動 → ハードドロップ」と一致します。

```js
{ id: "my_opener", name: "自作開幕", category: "開幕", pc: false,
  desc: "説明…",
  steps: [ { piece:"O", rot:0, col:0 }, { piece:"I", rot:1, col:9 } ] }
```

- `rot`: 0=出現 / 1=右回転 / 2=180 / 3=左回転
- `pc: true` にすると「全消し成立」を検証パネルでチェックします。
- 途中でラインが揃う構成は検証で警告されます（最終手で揃える前提）。

## ローカルで動かす

```powershell
# Windows / PowerShell（このフォルダで）
powershell -NoProfile -ExecutionPolicy Bypass -File serve.ps1 -Port 8230
# → ブラウザで http://localhost:8230/ を開く
```

`index.html` を直接ダブルクリック（`file://`）でも動きますが、Service Worker（オフライン化）は
http(s) のときだけ有効になります（`file://` では無効化され、エラーは出ません）。

## GitHub Pages で公開

1. GitHub で空の Public リポジトリ `tetris` を作成（README等は追加しない）。
2. このフォルダの中身を**リポジトリのルート**に push（`index.html` がルートに来るように）。
3. リポジトリ → Settings → Pages → Source: `main` / `(root)` → Save。
4. 数分後 `https://<ユーザー名>.github.io/tetris/` で公開。

## 検証・修正履歴

公開前にエンジン/デコーダ/各モード/入力/保存・取込/同梱データを独立監査し、確定した不具合を修正：

- T-Spin: ハードドロップで `lastRotation` が消え、浮いて着地する **TST/T-Spinトリプル系が判定されない**問題を修正（3-cornerルールで最終位置にて判定）。
- 手順型ドリル: 誤配置すると誤ミノが盤面に固定され**盤面が永久破損**しヒントもズレる問題を修正（誤配置は固定せずやり直し）。
- テト譜手順(fsteps): **ライン消去数が常に0**でT-Spinラベル/PCが出ない問題、最終形が**未消去のまま表示**される問題を修正。
- JSONインポート: **無検証で既存を上書き・壊れたデータでクラッシュ**する問題を修正（形状検証＋上書き通知＋防御）。
- 入力: 回転/ハードドロップ/ホールド/Undo/Reset の**OSキーリピート連射**を抑止。左右キーを**離さず切替えるとDASが止まる**問題を修正。
- 表示: T-Spin/消去ラベルがPC・完了メッセージに**同フレームで上書き**され見えない問題を統合表示で修正。
- テト譜: 20段超ブロックの**無警告切り捨て**を明示エラー化。
- 同梱データ: `op4`（PCO Iホールド有）が `op3` と**同一データで「検証済」と偽表示**だったため、重複データを除去（未設定＝後から正しい形を登録可能）。
